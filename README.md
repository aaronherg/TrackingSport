<hr></hr> <div align="center" width="50"> <p><strong> 👨‍💻 CAPACIDADES Y OPERATIVIDAD 📊 </strong></p> </div> <div align="start"> <div align="center"><p><strong> 1.0 Registro de Partidos </strong></p> </div>

1.1 Ver detalle de partidos: En este apartado podrás observar detalladamente la información de cada partido, incluyendo equipos participantes y puntos anotados.

1.2 Método de ingreso de estadísticas: El sistema permite a los usuarios especificar el método para ingresar estadísticas, ya sea manualmente o a través de una API de datos deportivos.

1.3 Generar reportes de partidos: El sistema permite generar reportes detallados sobre el rendimiento de los equipos y jugadores en cada partido.

<div align="center"><p><strong> 2.0 Consultas de Estadísticas </strong></p> </div>

2.1 Consultar datos de jugadores: En este apartado se pueden consultar los datos específicos de los jugadores, como estadísticas de rendimiento y partidos jugados.

2.2 Consultar balance de equipos: Esta consulta permite observar datos destacados sobre el rendimiento general de un equipo, incluyendo victorias, derrotas y puntos anotados.

2.3 Consultar historial de estadísticas: Aquí se puede listar todas las estadísticas registradas de los jugadores y equipos hasta el momento.

2.4 Consultar rendimiento de jugadores: En este espacio se pueden observar todas las estadísticas individuales de los jugadores, analizando su desempeño en diferentes partidos.
<div align="center"><p><strong> 3.0 Panel de Control </strong></p> </div>

3.1 Gestión de equipos: El sistema permite gestionar los datos de los equipos y los jugadores que los componen, para mantener un control de las estadísticas. 

3.2 Gestión de partidos: El sistema permite gestionar la programación y resultados de los partidos, asegurando un registro preciso de los eventos.

3.3 Gestión de jugadores: El sistema permite gestionar los jugadores que previamente tendran vinculación con los diferentes equipos que los conforman.
<div align="center"><p><strong> 4.0 Reportes </strong></p> </div>

4.1 Reporte de jugadores destacados: El sistema permite generar reportes sobre los jugadores con mejores estadísticas hasta la fecha, destacando su rendimiento.

4.2 Reporte de rendimiento de equipos: El sistema permite generar reportes sobre el rendimiento de los equipos según las estadísticas registradas.

4.3 Reporte de historial de partidos: El sistema permite generar reportes sobre los partidos jugados, mostrando resultados y estadísticas.
<div align="center"><p><strong> 5.0 Ajustes </strong></p> </div>

5.1 Ajustes del sistema: En este apartado, el sistema permite definir los datos operacionales de la organización, se almacenaran parametros importantes para poder evaluar los niveles de los jugadores y demas operaciones importantes.

5.2 Parámetros estadísticos: En este apartado, el sistema permite definir los parámetros de cálculo que se utilizan en el sistema, como promedios y rangos de rendimiento.
